# Library Management System in C++

A simple Library Management System developed using C++. This project uses object-oriented programming and file handling to manage books efficiently.

## Features

- Add new books
- View all books
- Search books by title
- Issue books
- Return books
- File handling using `.txt` file
- Menu-driven program

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling

## File Used

```text
librarymanagementsystem.txt
```

This file stores all book records.

## How to Run

1. Compile the program:

```bash
g++ main.cpp -o library
```

2. Run the program:

```bash
./library
```

## Sample Menu

```text
===== Library Management System =====

1. Add Books
2. View Books
3. Search Book
4. Issue Book
5. Return Book
6. Exit
```
